PRIVACY POLICY FOR MOBILE APPLICATION "FLAMES"

Last updated: July 05, 2022

This privacy policy governs your use of the software application "FLAMES" (later referred as the/this “Application”) for Android™ mobile devices that was created by Singam Solutions.

The Application is a mobile Android application intended to calculate compatibility between individuals.

What information does the Application obtain and how is it used?

The Application does not require any user registration. The Application does not require the user to provide any personal information.

The only User Provided Data that this Application uses is the information you provide directly within the Application when checking compatibility (e.g., individual names).

NO details are stored locally on your mobile device, or within the Application private folder.

Automatically Collected Information

The Application does not collect any information automatically.

The Application does not share any information automatically with third parties.
